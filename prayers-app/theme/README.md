# פלטת צבעים — "אנו בנייך"

מקור אמת יחיד לצבעי המותג: [`colors.json`](./colors.json).
משם נמשכים גם `tailwind.config.js` (קלאסים) וגם `constants/theme.ts` (טוקנים ל-StyleSheet).
**אין לכתוב hex ידני בקומפוננטות** — להשתמש ב-`COLORS` / `THEME` או בקלאסים של Tailwind.

## הצבעים

| טוקן             | ערך       | מקור בלוגו           |
| ---------------- | --------- | -------------------- |
| `primary` (navy) | `#1B2A4A` | טקסט "אנו בנייך" כהה |
| `gold`           | `#C9A84C` | זהב / הילה           |
| `flame` (כתום)   | `#E07B2C` | הלהבה                |
| `teal` (טורקיז)  | `#2E8B84` | הידיים               |
| `surface`        | `#FAF6EF` | רקע קרם חמים         |
| `ink`            | `#1B2A4A` | טקסט                 |

לכל גוון יש `light` / `dark` (חוץ מ-`accent` שהוא alias ל-`gold`).

> ⚠️ הערכים מקורבים — נשלפו מהלוגו הזמני שהתקבל. לאחר קבלת קובץ הלוגו הסופי
> יש לדגום מחדש את ה-hex המדויקים ולעדכן את `colors.json` בלבד (שאר הקבצים נמשכים ממנו).
